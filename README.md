Tela de Login Moderna — HTML, CSS e JavaScript

Uma interface de login responsiva, elegante e funcional, construída usando HTML, CSS e JavaScript puro. Este projeto foi criado para demonstrar um fluxo de autenticação simples, com validação, animações e mensagens amigáveis ao usuário.

✨ Funcionalidades

✔ Design moderno com gradiente e sombras ✔ Formulário responsivo ✔ Validação de e-mail e senha ✔ Feedback visual (erro/sucesso) ✔ Botões de login social (Google e Facebook) ✔ Sistema “Lembrar-me” com localStorage ✔ Redirecionamento após login bem-sucedido ✔ Código organizado e fácil de aprender

🛠️ Tecnologias Utilizadas HTML5 – Estrutura da página CSS3 – Estilos, animações e layout responsivo JavaScript – Lógica, validação e interações

🔐 Sobre o Login Este projeto não usa banco de dados. A autenticação é apenas simulada com JavaScript, ideal para estudo ou para servir de base para um backend real futuramente.

📌 Melhorias Futuras (opcional) Conectar com API real de autenticação Criar página de recuperação de senha Criar página de cadastro Implementar OAuth verdadeiro (Google, Facebook…)
